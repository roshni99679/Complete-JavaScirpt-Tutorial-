Exceptions and Error Handling in Js

1.syntax errors(parsing errors_)

occurs at compile time in traditional programming lang but at interpret time in javascript

example:

window.print(; //inside script tag

2.Runtime errors (occurs druing execution -after compilation or interpretation )

example:

window.printme(); //inside script tag

runtime error - calling a fn that does not exist

3.Logical errors(most difficult type of errors)

you cannot catch these errors as they are not a result of runtime or syntax errors

thay occur when you make a mistake in the logic that drives your script and you do not get the result you expected
