Introduction to Cookies (HTTP Cookie /internet cookie)

js can read create modify and delete cookie that apply to the current webpage

cookies are plain text data record of 5-variable-length fields-

1.Expires: The date when cookie will expire . if this is blank , the cookie will expire when the visitor quits the browser

2.Domain:The domain name of your site

3.path:The path to directory or webpage that set the cookie

4.Secure:if this field contains word "secure", then the cookie may be only retrieved with a secure server

3.Name=Value:cookies are set and retrieved in the form of key-value pairs

cookies are created by web browsers and sent and stored in web server. Cookies have a unique id by which they can be retrieved

Types of cookies

1.Session Cookies

these are stored in temporary memory and is not retained if the browser is closed

2.Persistent cookies

also known as stored cookies . it is stored for a fixed peiod of time on computer,and is not deleted when the browser is closed

Creating a cookie

in js cookies are modified with the help of cookie property of document object

with the help of document.cookie property js can read , create,modify and erase the cookies
